# 房产大师 - 核心需求文档

## 项目概述

**名称**: 房产大师 (Fangdai Master)
**目标用户**: 海外中国人
**核心价值**: 帮助海外中国人管理房地产和规划税务

## 功能模块

### 1. 税务规划 💰

#### 1.1 税务咨询
- 用户可以向 AI 助手提问关于海外房产税务问题
- 支持个人所得税、房产税、资本利得税等
- 提供专业、合规的税务建议

#### 1.2 税务方案规划
- 根据用户房产情况生成个性化税务方案
- 计算预估税款
- 优化税务策略

#### 1.3 税款缴纳指导
- 申报流程指导
- 截止日期提醒
- 缴纳方式说明

### 2. 房产管理 🏠

#### 2.1 房产信息管理
- 添加/编辑/删除房产
- 记录房产基本信息（地址、面积、购买价格等）
- 房产估值追踪

#### 2.2 租金收益管理
- 记录租金收入
- 统计年度收益
- 收益报表生成

#### 2.3 多房产管理
- 统一管理多个房产
- 分类筛选
- 总资产统计

### 3. 证件管理 📄

#### 3.1 证件数字化
- 房产证扫描存储
- 购房合同存档
- 身份证明文件

#### 3.2 证件提醒
- 证件到期提醒
- 续期时间线
- 重要日期日历

#### 3.3 产权信息
- 产权年限
- 产权变更记录
- 法律文件管理

### 4. 政策解读 📚

#### 4.1 最新政策推送
- 房产相关法规更新
- 税务政策变化
- 移民政策影响

#### 4.2 政策解读
- 专业分析解读
- 影响评估
- 合规建议

### 5. AI 智能助手 🤖

#### 5.1 智能问答
- 7x24 小时在线
- 个性化建议
- 历史记录保存

#### 5.2 场景化服务
- 购房前咨询
- 持有期间管理
- 出售规划

## 数据模型

### 用户 User
- id, openid, nickname, avatar, created_at

### 房产 Property
- id, user_id, name, address, type, purchase_date, purchase_price, current_value, images, created_at

### 税务记录 TaxRecord
- id, user_id, property_id, year, tax_type, amount, status, due_date, paid_date, created_at

### 证件 Document
- id, user_id, property_id, type, name, number, issue_date, expiry_date, image_url, created_at

### AI 对话记录 ChatHistory
- id, user_id, messages, created_at

## 优先级

### P0 - 必须有
1. 用户登录/注册
2. 房产基本信息管理（CRUD）
3. AI 税务咨询入口
4. 税务记录查看

### P1 - 重要
1. 证件管理
2. 租金收益记录
3. AI 方案规划
4. 政策推送

### P2 - 增强
1. 证件扫描 OCR
2. 智能提醒
3. 数据报表
4. 多语言支持

## 页面规划

1. **首页** - 功能入口 + AI 助手快捷入口
2. **房产列表** - 所有房产卡片展示
3. **房产详情** - 单个房产完整信息
4. **添加房产** - 表单录入
5. **税务中心** - 税务记录列表 + AI 咨询
6. **证件库** - 证件列表 + 扫描
7. **政策资讯** - 文章列表
8. **我的** - 用户设置

## 目标用户痛点

1. **信息分散** - 房产、证件、税务信息分散各处
2. **政策不明** - 海外房产政策复杂，难以理解
3. **税务复杂** - 各国税制不同，不知道怎么交税
4. **证件遗失** - 重要文件容易丢失或过期
5. **缺乏规划** - 不知道如何优化税务和资产配置
